Comandos Git

CLONANDO:

Inicialização das atividades:

  1. git fetch --all

    2. (sincroniza o repositorio local
    com o repositorio remoto)

  2. git pull

    3. (Faz o download das atualizações
    do servidor remoto)

Finalização das atividades:

  1. git status

    2. (Lista as modificações)

  2. git add .

    3. (Adiciona as modificações
    no servidor local)

  3. git commit -m "[texto]"

    4. (Monta as modificações para
    envio ao servidor remoto)

  4. git push

    5. (Faz upload das modificões
    do servidor local para o
    servidor remoto)
